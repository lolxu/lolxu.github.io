    Hi there! I'm David. I am an undergraduate senior student currently studying at the University of Southern California, majoring in Computer Science with a concentration in games.

    Projects

    Oasis Blitz

    Gameplay Engineer on Oasis Blitz - A unique drill-type platformer implemented with Unity. We used Hiearchical Statemachine as the main character system. My work includes XML save/load, Character and Enemy gameplay mechanics, Optimization, and more. This is my capstone project at USC.

    DirectX Game Engine

    A DirectX11 game engine programmed from scratch using C++, HLSL, Multi-threading for animation jobs, and a custom asset pipeline. It features Skeletal Animation, normal mapping, multiple render targets (support post-processing), and basic 3D collision.

    UE5 Multiplayer FPS

    A class project made using UE5 C++ and blueprint programming. I created a multiplayer FPS template with functional replication on both Listen Server and Dedicated Server.

    JARHEAD - UE5

    A WIP UE5 project from me that tries to recreate a tactical third person shooter experience from a helmet-cam perspective

    Simply Rotate

    A fun little physics-based puzzle game where you rotate the entire level to solve problems. A game made with Unity.

    Getaway

    A mobile game made in Unity where you swipe the screen to evade cops.

    Untitled Platforming Game

    Submission to GMTK 2023 Game Jam: Roles Reversed. It offers an interesting reverse on platforming.

    C++ SDL Projects

    A series of game projects made using the SDL library and C++. Here, I practiced making games from scratch using C++ and Actor / Component based structure.

    Game Prototypes

    Several game prototypes made with different constraints and limited scopes. These prototypes are made with Unity and C#.

    Tencent Games Summer Open Course Projects
