    • UE5 Multiplayer FPS

    A fully online multiplayer created as a class project. I have implemented basic online replication using UE C++. I used Server RPCs and other methodologies to achieve a multiplayer FPS envrionment. Everything from animations, ammo pickups, firing, scoring, and even decals are implemented to be fully replicated using a mix of C++ and Blueprint Programming. The multiplayer FPS template can run on a Dedicated Server or a Listen Server.

    I utilized subsystems to account for integration to Steam, and I have fully implemented the ability to join a game through Steam using the friend system. I also used the GameInstanceSubSystem to achieve saving and loading. The decals that player can spray in the game world can be saved and loaded using this subsystem.

    This multiplayer FPS template is highly usable with numerous UI implementations as well. Players can vote to begin the game. I have also implemented a full chat system in this project that is replicated through the use of replications in PlayerStates.
