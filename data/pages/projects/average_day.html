    • An Average Day

    This is a class project created for the Intermediate Game Development course. We had a semester to work in pairs to prototype, develop and publish a working game on Itch.io. We created a puzzle game using Unity and C# that takes a interesting twist on classic Sokoban style puzzle gameplay. We used Sprints and burn-down charts to organize the workflow of our project. For playtesting and balance, we also used a custom metrics tracker that generate each time someone has played the Unity build.

    You play as a kid who is trying to get up and go to school in time, but there are plenty weird obstacles in the way. The player need to plan ahead because when the character return home at night, the boxes you pushed will become immobile, and you only have limited steps to take.

    We used several different libraries (including DOTween) in Unity to create this game. We also collaborated with students from Berklee College of Music to create the soundtrack for the game.
