    • DirectX Game Engine

    This is a DirectX11 Game Engine developed from my ITP485 - Game Engine Programming class.

    Implementing this game engine allowed me to learn the basics of using DirectX buffers, how rendering works in general, and how basic functionalities like lighting, animations, and physics collisions can be implemented in a real game engine.

    I used HLSL to create different shaders such as the Phong Lighting shader, Skinning shader for skeletal animations, a Toon shader for the character, a Normal map shader, a Gaussian blur shader for implementing bloom with multiple Render Targets, and more basic vertex and pixel shaders for unlit rendering.

    I also implemented an asset import pipeline that reads information from JSON with a specific format to import vertices from meshes, animation data, materials with textures, and level data. I further improved this asset pipeline to support a Actor-Component based structure. For example, an actor can have a lighting components, and this importer will import and create components according to JSON.
