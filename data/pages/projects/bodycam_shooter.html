    • Bodycam Shooter UE5 Project

    This is a work-in-progress project that I am taking on individually. With the recent hype of Unrecord and the popularization of bodycam style gameplay, I want to try and create something like them on my own using Unreal Engine 5 with a heavy mix of Blueprint and C++ programming.

    I am creating a True First Person Tactical shooter experience that will use heavily stylized bodycam style gameplay. The bodycam effect is quite simple to recreate. I got a pretty realistic implementation by using the UE5 post-processing materials with lens distortion and other post-processing effects. Then, I created a automatic Depth of Field focusing mechanic using a combination of Blueprint and C++. However, the True First Person setup will be quite complex. I have already applied Inverse Kinematics for weapon holding, weapon sway, and procedural recoil as displayed in the thumbnail.

    Currently, I am working on finishing the True First Person template. I will try to add physical animations with ragdoll effects later to create realistic ragdoll for this project.
