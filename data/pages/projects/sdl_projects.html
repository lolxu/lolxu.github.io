    • SDL-Library Projects

    This is a series of 2D and 3D game projects made with the SDL library in C++. These projects are class projects from ITP380 - Video Game Programming. In the class, we learned how to create a working game engine pipeline using the SDL kit (from processing user input to update functions using deltatime to generating output using SDL rendering)

    We learned the basics of vector math, lerp functions and quarternions for rotating objects in 3D. Also, we delved into simple AI making with A* pathfinding (for the Zelda project) and a custom pathfinding algorithm using path nodes, similar to BFS (for the PacMan project).
