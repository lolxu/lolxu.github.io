    • Tencent Games Summer Open Course Projects

    These prototypes are created from the Tencent Games 2022 Summer Open Course.I was lucky to get in to this program after a series of coding tests and interviews. Gameplay Engineers from Tencent Games hosted virtual online sessions that would teach the basics of Unreal Engine 4 to college students.

    During the summer course, I learned how to program basic FPS logic using Blueprint and C++ programming. I learned how to create and blend animation using the animation blueprint system in UE4 to create fluent FPS animations. I also learned how to create custom materials and UI. Mostly importantly, I practiced how to create AI logic using the blackboard / behavior tree system, and I implemented a multiplayer FPS shooter template in UE4 using replication and RPC.
