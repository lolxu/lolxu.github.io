    • Untitled Platforming Game

    Untitled Platforming Game is a submission to the GMTK-2023 game jam made using Unity and C#. The theme of this year was Roles-Reversed. Participants had only 48 hours to make a playable game that fits the theme. I started a bit late... When designing the game, I thought it would be an interesting twist if the players, instead of controlling the platofrming character, would be controlling the platforms.
